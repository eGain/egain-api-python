## get_topic_breadcrumb_for_article

## Overview
  * Use this API to retrieve the topic breadcrumb for an article in a portal. A breadcrumb shows the hierarchical path from the root topic to the article's primary topic.


### Example Usage

<!-- UsageSnippet language="python" operationID="getTopicBreadcrumbForArticle" method="get" path="/portals/{portalID}/articles/{articleID}/breadcrumb" -->
```python
from egain_api_python import Egain
import os


with Egain(
    access_token=os.getenv("EGAIN_ACCESS_TOKEN", ""),
) as egain:

    res = egain.portal.topic.get_topic_breadcrumb_for_article(accept_language="en-US", portal_id="PROD-1000", article_id="PROD-2996", language="en-US")

    # Handle response
    print(res)

```

## getchildtopics

## Overview
  The Get Immediate Child Topics API retrieves details about a topic and it's immediate child topics. The <code>$level</code> attribute determines how deep the topic hierarchy should go, or how many child topics of a topic are returned in the response.


### Example Usage

<!-- UsageSnippet language="python" operationID="getchildtopics" method="get" path="/portals/{portalID}/topics/{topicID}" -->
```python
from egain_api_python import Egain
import os


with Egain(
    access_token=os.getenv("EGAIN_ACCESS_TOKEN", ""),
) as egain:

    res = egain.portal.topic.getchildtopics(accept_language="en-US", portal_id="PROD-1000", topic_id="PROD-1069", search_profile_id="959500000204621", level=-1, language="en-US", topic_additional_attributes=[
        "modifiedBy.userName",
    ], custom_additional_attributes="country_name", pagenum=1, pagesize=10)

    assert res is not None

    # Handle response
    print(res)

```

## getalltopics

## Overview
  The Get All Topics API retrieves the topic tree for a portal. The <code>$level</code> attribute determines how deep the topic hierarchy should go, or how many child topics of a topic are returned in the response.


### Example Usage

<!-- UsageSnippet language="python" operationID="getalltopics" method="get" path="/portals/{portalID}/topics" -->
```python
from egain_api_python import Egain
import os


with Egain(
    access_token=os.getenv("EGAIN_ACCESS_TOKEN", ""),
) as egain:

    res = egain.portal.topic.getalltopics(accept_language="en-US", portal_id="PROD-1000", level=-1, language="en-US", topic_additional_attributes=[
        "modifiedBy.userName",
    ], custom_additional_attributes="country_name", pagenum=1, pagesize=10)

    assert res is not None

    # Handle response
    print(res)

```
